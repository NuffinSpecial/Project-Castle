About the Project

{{ icon('castle', 64, 'icon--brand') }}

Project Castle (Contextual ASL Translator) links English sentences to approximate ASL gloss using NLP, then plays crowdsourced sign videos contributed by the community.

The pipeline tokenises and analyses English (spaCy when available), applies gloss rules and multi-word expressions, then maps each gloss token to a community video when one exists.

Built by NuffinSpecial. Every sign you submit on the Submit Translation page is added to the live catalog immediately.
